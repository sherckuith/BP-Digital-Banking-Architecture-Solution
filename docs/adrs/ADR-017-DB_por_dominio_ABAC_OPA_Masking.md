# ADR-017: DB por dominio + ABAC/OPA + masking

## Status
Accepted

## Context
[Contexto del problema arquitectónico que se resuelve con esta decisión]

## Decision
DB por dominio + ABAC/OPA + masking

## Consequences
1. Limita movimiento lateral de datos sensibles (principio least privilege).
2. Masking/Tokenization cumple con LOPDP y facilita auditorías.
