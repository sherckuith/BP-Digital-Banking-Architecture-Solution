# ADR-013: Observabilidad con OpenTelemetry

## Status
Accepted

## Context
[Contexto del problema arquitectónico que se resuelve con esta decisión]

## Decision
Observabilidad con OpenTelemetry

## Consequences
1. Evita vendor lock-in y permite tracing end-to-end.
2. Mejora MTTR mediante correlación de logs, métricas y trazas.
