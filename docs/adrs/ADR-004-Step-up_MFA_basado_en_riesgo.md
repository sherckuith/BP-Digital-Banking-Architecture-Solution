# ADR-004: Step-up MFA basado en riesgo

## Status
Accepted

## Context
[Contexto del problema arquitectónico que se resuelve con esta decisión]

## Decision
Step-up MFA basado en riesgo

## Consequences
1. Reduce fraude sin degradar UX (solo cuando monto > MTU, nuevo beneficiario, etc.).
2. Cumple con CNBV 12-E que obliga MFA reforzado para transacciones de riesgo.
