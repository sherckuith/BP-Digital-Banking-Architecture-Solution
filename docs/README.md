# Documentación del Proyecto

Este directorio centraliza toda la documentación técnica, de cumplimiento y operativa del proyecto.

## Contenido

*   **adrs/**: Registro de decisiones de arquitectura (ADRs). Documenta las decisiones clave, su contexto, consecuencias y estado.
*   **compliance/**: Documentación relativa al cumplimiento de normativas locales (LOPDP, CNBV) e internacionales (PCI-DSS, ISO 27001).
*   **observability/**: Estrategias de monitoreo, trazas y logs utilizando OpenTelemetry.
*   **README.md**: Este archivo.

Para una visión técnica detallada, consulte los diagramas en la carpeta `../diagrams/`.
